# Phone Dialer +
This is a simple dialer app that uses the system dialer to dial numbers and is usefull for elderly people who think the normal system dialer is complicated


## Features
- Easy to use

- Lightweight, doesn't take much resources

- Big bold font for textbox

- Bright green color for the call button

- Open source


## Installation
- Open the APK file

- If it prompts to enable unknown sources, enable it

- Click "Install"


## Usage
- Click on the textbox to focus the textbox

- Enter number (eg. +11234567890)

- Click "Call" to dial the number
